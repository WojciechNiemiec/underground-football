# Purpose of this directory

This directory contains Bruno collections to demonstrate Football Underground API.
Bruno is a fully local and Git-native solution for API testing and sharing.

Do not update the contents of [underground-football](football-underground) directory manually.

More info: https://www.usebruno.com

## Usage

1. Download and install Bruno from https://www.usebruno.com/downloads
2. Import [underground-football](football-underground) collection from this directory using 3-dot menu in Bruno app.
3. Do not store any IDs or tokens in the collection. They should be managed dynamically using environment variables.
4. Test, explore and collaborate!
